#  MERN BLOG app

A blog with all CRUD functinality

## Usage

Add your environment Variable in .env file

### Install dependencies

```
# Backend dependency - Express Js - Node 
npm install

# Frontend dependency - React and Redux 
cd frontend
npm install
```

### Run Server

```
npm run server
```
